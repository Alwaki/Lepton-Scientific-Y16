## Getting Started

Note: Tested on Ubuntu 22.04, using Lepton 3.5 and 3.1R, and Purethermal 3.0 board.

### Dependencies

* OpenCV2
* Numpy
* v4l2-ctl

### Executing program

from the appropriate folder, execute the code in terminal:

```
python camera.py
```
